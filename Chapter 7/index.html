‚è´ Increment (++), Decrement (--) Behavior

  • x++ ‚Üí Returns original value, then increments
  • ++x ‚Üí Increments first, then returns new value
  • x-- ‚Üí Returns original value, then decrements
  • --x ‚Üí Decrements first, then returns new value
  • Example:
    let a = 5;
    console.log(a++); // 5 (then becomes 6)
    console.log(++a); // 7 (was 6, then incremented before return)
