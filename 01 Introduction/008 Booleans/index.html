üìò Important Notes on Booleans & Comparison Operators

  • Boolean Values: Only two ‚Üí true and false
  • Truthy & Falsy: Values like 0 ""null undefined NaN false are falsy. Everything else is truthy.
  • String Methods: Methods like .includes() return Boolean values.
  • Comparison Operators: Use == != === !==> < >= <=
  • Loose Comparison (==): Converts data types before comparing. Can lead to unexpected results.
  • Strict Comparison (===): Compares both value and type. Safer and recommended.
  • Case-sensitive: Comparisons between strings are case-sensitive "hello" !== "Hello"
  • Lexicographic String Comparison: Strings are compared character-by-character by Unicode value.
