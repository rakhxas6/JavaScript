Primitive Types & Reference Types

ðŸ“˜ Primitive Types vs Reference Types in JavaScript

  • Primitive Types: Stored in the "STACK". They are copied by value.
  • The Types include: String, Number, Boolean, undefined, null, Symbol, BigInt
  • Reference Types: Stored in the "HEAP". They are copied by reference.
  • The Types include: Object, Array, Function, Date, RegExp, Custom objects
  • Stack: Fast, small memory where primitive data is stored.
  • Heap: Larger memory area where objects and functions are stored.
  • Immutable: Primitive values cannot be changed (e.g., strings are immutable).
  • Mutable: Reference values (like arrays and objects) can be changed.
