What is Class Inheritance?

Class inheritance is a mechanism in object-oriented programming that allows a class (called a subclass) to inherit properties and methods from another class (called a superclass).

This promotes code reusability and establishes a hierarchical relationship between classes.

  • Subclass: The class that inherits from another class.
  • Superclass: The class from which properties and methods are inherited.
  • Overriding: Providing a new implementation for a method in the subclass that already exists in the superclass.
