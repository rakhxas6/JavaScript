What is the Prototype Model?

The prototype model is a programming paradigm used in JavaScript and other prototype-based languages. It allows objects to inherit properties and methods from other objects, rather than from classes as in classical inheritance.

This promotes code reusability and establishes a more flexible relationship between objects.

  • Prototype: The object from which other objects inherit properties and methods.
  • Delegation: The process by which an object can access properties and methods of another object through the prototype chain.
  • Overriding: Providing a new implementation for a method in a derived object that already exists in the prototype.

Prototypes

  • Every object in JS has a prototype
  • The prototype contain all the methods for that object type.
