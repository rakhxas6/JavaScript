JavaScript Data Types

Getting started with numbers

➕ String + Number Behavior

  • "5" + 5 → "55" – Concatenation
  • 5 + "5" → "55" – String wins
  • "5" - 2 → 3 – String converted to number
  • "hello" - 5 → NaN – Invalid subtraction
  • "5" + 5 - 5 + "5" → "505" – Mixed behavior

⏫ Increment (++), Decrement (--) Behavior

  • x++ → Returns original value, then increments
  • ++x → Increments first, then returns new value
  • x-- → Returns original value, then decrements
  • --x → Decrements first, then returns new value
  • Example:
    let a = 5;
    console.log(a++); // 5 (then becomes 6)
    console.log(++a); // 7 (was 6, then incremented before return)
